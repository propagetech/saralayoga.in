# Build guide

This project has **no build step**. Edit HTML, `src/css/styles.css`, and `src/js/*.js` directly.

**Local preview:** use a static server (e.g. Live Server in the editor, or `npx --yes serve . -l 3000` from the project root) so `/src/...` URLs resolve.

For deployment, upload the repository or connect it to a static host with an **empty** build command. See [README](../README.md) and [DEPLOYMENT](./DEPLOYMENT.md).
