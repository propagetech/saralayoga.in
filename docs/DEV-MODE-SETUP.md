# Development

This project does **not** use npm, Tailwind CLI, or any Node-based build.

- Edit **HTML** in the repository root, **CSS** in `src/css/styles.css`, and **JS** in `src/js/`.
- Preview locally: use a **Live Server** style extension in Cursor/VS Code, or from the repo root run `npx --yes serve . -l 3000` (requires Node.js for that command only) and open `http://localhost:3000`.

See [README](../README.md) for structure and deployment.
